# 📺 Play Now QuickNav V2
**Profesjonalny "pilot" klawiaturowy do platformy PLAY NOW.**

Masz dość celowania myszką w małe ikonki kanałów? Ten program pozwala na błyskawiczną nawigację po kanałach TV bezpośrednio z klawiatury.

## 🚀 Główne Funkcje
* **Szybki Wybór:** Aktywuj pilot prawym klawiszem **Shift** i wybierz numer kanału.
* **Dwie Architektury:** Dedykowane wersje **x64** (nowe systemy) oraz **x86** (starsze komputery).
* **Pełna Oprawa:** Obsługa logotypów stacji oraz zintegrowane EPG.
* **Open Source:** Kod źródłowy dostępny do wglądu – 100% transparentności.

## 📥 Pobieranie
Gotowe paczki ZIP znajdziesz w sekcji **Releases** po prawej stronie strony głównej repozytorium. Pobierz wersję pasującą do Twojego systemu, wypakuj i odpal plik `.exe`.

## 🛡️ Ważne: Informacja o Antywirusach
Program został stworzony w języku **AutoHotkey**. Ze względu na brak płatnego, cyfrowego certyfikatu, system **Windows Defender** oraz inne antywirusy mogą oznaczyć plik jako "niebezpieczny" (tzw. False Positive).
> **Zapewniam: program jest w pełni bezpieczny.** Kod źródłowy jest jawny – możesz go samodzielnie przejrzeć w tym repozytorium przed uruchomieniem.

## ⌨️ Instrukcja Obsługi
1. Uruchom program (pojawi się ikona w zasobniku systemowym).
2. Naciśnij i przytrzymaj **Prawy Shift**.
3. Wybierz numer kanału na klawiaturze (np. 1, 2, 3...).
4. Program automatycznie przełączy widok w przeglądarce.
