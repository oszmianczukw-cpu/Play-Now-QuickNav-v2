# Play-Now-QuickNav-v2
Szybki nawigator kanałów dla PLAY NOW. Obsługa skrótami klawiszowymi (Prawy Shift + numer), wsparcie dla EPG i logotypów stacji. Dostępne wersje x64 i x86. Kod źródłowy w 100% otwarty (AutoHotkey v2). Uwaga: Windows Defender może zgłaszać False Positive ze względu na brak certyfikatu – program jest całkowicie bezpieczny
